# Contributing Guidelines

Thank you for your interest in contributing to the Supply Chain Risk & Strategic Petroleum Reserve (SPR) Optimization Platform!

## How to Contribute

1. **Fork the Repository**: Create your own copy of the repository by clicking the Fork button on GitHub.
2. **Clone your Fork**:
   ```bash
   git clone https://github.com/Pratik8035/HackthonProject.git
   cd HackthonProject
   ```
3. **Create a Feature Branch**:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Make Your Changes**: Ensure both backend tests (`python integration/test_integration.py` in `Backend/HackthonProject`) and frontend build (`npm run build` in `Frontend`) pass cleanly.
5. **Commit Your Changes**:
   ```bash
   git commit -m "feat: add your detailed description"
   ```
6. **Push to Your Fork**:
   ```bash
   git push origin feature/your-feature-name
   ```
7. **Open a Pull Request**: Submit a Pull Request targeting the `main` branch with a clear summary of your changes.
